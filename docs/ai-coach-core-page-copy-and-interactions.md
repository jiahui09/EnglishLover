# EnglishLover AI Coach 核心页面文案与交互稿

> 文档定位：目标产品方案 / 核心页面文案与交互稿
> 页面范围：首页、单词页、阅读页、教练页。
> 范围边界：仅描述页面文案、内容区块、交互状态和反馈；不涉及技术架构、接口实现、数据库或部署。
> UI 细化参考：`docs/ai-coach-ui-design-guidelines.md`；全局设计决策源：`DESIGN.md`。

## 1. 全局文案原则

### 1.1 语气

- 像私人教练，不像冷冰冰的数据面板；
- 鼓励但不夸张；
- 具体但不啰嗦；
- 指出问题时给出下一步行动；
- 不承诺“快速提分”“必然掌握”等未验证结果。

### 1.2 推荐表达

| 场景 | 推荐文案 | 避免文案 |
|---|---|---|
| 错误反馈 | 这道题暴露出推断题弱项，建议稍后重点复盘 | 你做错了 |
| 学习建议 | 今天先复习 8 个薄弱词，再做一篇阅读 | 继续努力 |
| AI 生成失败 | 暂时无法生成新文章，可以先使用预置阅读继续练习 | 系统错误 |
| 学习不足 | 完成一次单词和阅读后，我会给出更准确的建议 | 暂无数据 |

## 2. 首页：今日学习

### 2.1 页面目标

首页负责回答用户最重要的问题：

> 我今天该怎么学？

### 2.2 页面结构

```text
首页 / 今日学习
├── 顶部问候
├── 今日总目标
├── 单词任务卡
├── 阅读任务卡
├── 教练建议摘要
└── 今日完成进度
```

### 2.3 首屏文案

#### 标题

```text
今天的英语学习计划
```

#### 副标题

```text
先把近期单词过一遍，再用它们完成一篇考研英语阅读。
```

#### 今日目标卡

```text
今日目标
完成 1 组单词复习 + 1 篇阅读理解
预计用时 15—25 分钟
```

### 2.4 单词任务卡

#### 默认状态

```text
单词输入
复习近期词，准备生成今日阅读

今日建议：
新词 8 个 · 复习 12 个 · 薄弱词 4 个

[开始单词训练]
```

#### 已完成状态

```text
单词输入已完成
你刚刚完成了 20 个词，其中 4 个需要回炉。

下一步：用这些词生成一篇考研阅读。

[生成阅读]
```

#### 数据不足状态

```text
还没有近期单词
先学习 5 个考研核心词，我会用它们生成第一篇阅读。

[开始第一组单词]
```

### 2.5 阅读任务卡

#### 可生成状态

```text
阅读输出
根据你近期学习的词生成考研英语阅读理解。

将使用：
context · assume · significant · evaluate · maintain

题型：
主旨题 · 细节题 · 推断题 · 词义题

[生成今日阅读]
```

#### 未完成单词状态

```text
阅读任务待解锁
建议先完成一组单词训练，阅读会更贴合你今天的学习内容。

[先去背词]
[直接使用系统推荐词]
```

#### 已完成状态

```text
阅读任务已完成
正确率 75%
推断题还有提升空间。

[查看解析]
[看教练建议]
```

### 2.6 教练建议摘要

#### 有学习记录

```text
教练建议
你今天的词汇识别不错，但推断题错了 2 道。下一轮建议少量复习薄弱词，再练 1 篇推断题更明显的阅读。

[查看完整建议]
```

#### 无学习记录

```text
教练建议
完成一次单词训练和阅读练习后，我会根据你的表现给出更具体的学习建议。

[开始今日学习]
```

### 2.7 交互规则

| 用户操作 | 页面反馈 |
|---|---|
| 点击“开始单词训练” | 进入单词专注模式 |
| 点击“生成今日阅读” | 进入阅读生成状态，并在生成成功或使用预置阅读后保持阅读专注模式 |
| 点击“直接使用系统推荐词” | 生成不依赖近期词的入门阅读 |
| 点击“查看完整建议” | 进入教练页 |
| 今日任务全部完成 | 首页显示完成状态和明日建议摘要 |

### 2.8 首页完成状态

```text
今日学习已完成
你完成了单词输入和阅读输出。明天会优先复习今天暴露出的 4 个薄弱词。

[查看今日总结]
[再做一篇阅读]
```

## 3. 单词页：单词训练

### 3.1 页面目标

单词页负责完成知识输入，并为阅读生成提供近期词基础。

### 3.2 页面结构

```text
单词训练
├── 专注模式顶部：返回 / 本轮标题 / 轻量进度
├── 本轮说明
├── 单词卡片
├── 掌握反馈按钮
├── 进度条
└── 本轮结果
```

### 3.3 本轮说明文案

```text
本轮单词训练
已进入专注模式。请先看当前词，再按真实掌握情况选择，不需要追求一次全对。

20 个词 · 预计 6 分钟
这些词会用于稍后的考研阅读生成。
```

### 3.4 单词卡片文案

#### 正面

```text
significant

先回忆它的意思，再点击查看释义。

[查看释义]
```

#### 背面

```text
significant
adj. 重要的；显著的

例句：
The policy had a significant impact on economic growth.

考研语境提示：
常用于描述影响、变化、差异是否“重要或显著”。
```

### 3.5 掌握反馈按钮

```text
[认识] [模糊] [不认识]
```

按钮解释：

| 按钮 | 含义 | 后续影响 |
|---|---|---|
| 认识 | 能准确理解词义 | 正常进入近期词池 |
| 模糊 | 见过但不稳定 | 阅读中优先复现 |
| 不认识 | 无法回忆或理解 | 加入薄弱词，后续回炉 |

### 3.6 进度文案

```text
第 6 / 20 个
本轮已标记：认识 3 · 模糊 2 · 不认识 1
```

### 3.7 本轮结果页

#### 标题

```text
本轮单词完成
```

#### 摘要

```text
你完成了 20 个词：
认识 12 个 · 模糊 5 个 · 不认识 3 个
```

#### 说明

```text
接下来我会优先把“模糊”和“不认识”的词放进阅读语境中，帮助你在文章中重新识别它们。
```

#### 行动按钮

```text
[用这些词生成阅读]
[稍后再做阅读]
```

### 3.8 空状态

```text
还没有可训练的单词
你可以先从考研核心词开始，完成第一组后即可生成阅读练习。

[开始 5 个核心词]
```

### 3.9 交互规则

| 用户操作 | 页面反馈 |
|---|---|
| 进入单词训练 | 隐藏底部导航、学习画像和历史统计，只保留返回、进度、当前词和反馈按钮 |
| 查看释义 | 卡片翻面展示释义、例句、语境提示 |
| 点击认识 | 进入下一个词，记录为已掌握倾向 |
| 点击模糊 | 进入下一个词，记录为需要复现 |
| 点击不认识 | 进入下一个词，记录为薄弱词 |
| 完成本轮 | 展示本轮结果，退出单词专注模式，并以“用这些词生成阅读”为主行动 |
| 未完成时返回 | 显示退出确认，说明本轮进度会保留或稍后继续 |

## 4. 阅读页：AI 考研阅读训练

### 4.1 页面目标

阅读页负责把单词输入转化为语境化输出，通过文章和题目检验用户是否真正理解近期词。

### 4.2 页面结构

```text
阅读训练
├── 专注模式顶部：返回 / 阅读或题目进度 / 当前主行动
├── 生成说明
├── 阅读文章
├── 目标词提示
├── 练习题
├── 提交结果
└── 解析与回流
```

### 4.3 阅读生成前

#### 标题

```text
生成今日考研阅读
```

#### 说明文案

```text
我会根据你近期学习的单词生成一篇考研英语级别的阅读理解，并配套练习题。开始阅读后会进入专注模式，只保留正文、题目和当前行动。
```

#### 目标词展示

```text
本次重点词：
significant · context · evaluate · maintain · assume
```

#### 难度说明

```text
难度：考研英语基础—中等
预计阅读时间：8 分钟
题目数量：4 题
```

#### 按钮

```text
[开始生成]
[更换一组目标词]
```

### 4.4 生成中状态

```text
正在生成阅读
我正在把你的近期单词组织进一篇考研英语阅读中。

这通常需要一点时间。生成失败时，你也可以使用预置阅读继续练习。
```

### 4.5 AI 失败状态

```text
暂时无法生成新文章
可能是 AI 服务暂时不可用。你可以重试，或先使用一篇预置考研阅读继续完成今日训练。使用预置阅读后仍会进入阅读专注模式。

[使用预置阅读]
[重试生成]
```

### 4.6 阅读文章区

#### 顶部信息

```text
考研阅读训练
已进入专注模式
主题：Education and Personal Growth
难度：考研英语中等
目标词：5 个
```

#### 目标词提示

```text
阅读时留意这些词在文章中的含义：
significant · context · evaluate · maintain · assume
```

#### 文章操作

```text
[隐藏目标词提示]
[查看词汇提示]
[开始答题]
```

### 4.7 练习题区

#### 题目标题

```text
Question 1 / 4 · Main Idea
```

#### 题型中文提示

```text
主旨题：先判断文章整体讨论的问题，不要只看某一句话。
```

#### 选项提交

```text
[A] ...
[B] ...
[C] ...
[D] ...

[下一题]
```

最后一题：

```text
[提交答案]
```

### 4.8 结果页

#### 标题

```text
阅读完成
```

#### 总览

```text
正确率 75%
4 题中答对 3 题
```

#### 分项表现

```text
主旨题：正确
细节题：正确
推断题：错误
词义题：正确
```

#### 目标词表现

```text
目标词回顾
你能正确理解 significant 和 maintain。
context 在词义题中表现不稳定，建议加入下一轮复习。
```

### 4.9 单题解析文案模板

```text
你的答案：B
正确答案：C

解析：
这道题考查推断能力。文章第二段并没有直接说明作者反对该观点，而是通过 “however” 后的转折表达保留态度，因此 C 更符合作者立场。

暴露问题：
你可能忽略了转折关系。

下一步建议：
复盘文章中的 however / although / yet 等逻辑连接词。
```

### 4.10 阅读后行动

```text
[查看教练建议]
[把薄弱词加入复习]
[再生成一篇]
[返回今日学习]
```

### 4.11 交互规则

| 用户操作 | 页面反馈 |
|---|---|
| 点击开始生成 | 进入生成中状态 |
| 生成成功 | 进入阅读专注模式，展示文章、目标词和开始答题按钮 |
| 生成失败 | 在当前阅读流程内提供预置阅读和重试，不跳回首页 |
| 使用预置阅读 | 进入阅读专注模式，并在教练建议中标注本次基于预置阅读 |
| 提交题目 | 展示正确率、解析、薄弱词和题型表现 |
| 点击加入复习 | 将薄弱词标记为后续复习对象 |

## 5. 教练页：私人学习建议

### 5.1 页面目标

教练页负责把学习数据转化为行动建议，让用户知道下一步该怎么学。

### 5.2 页面结构

```text
教练建议
├── 今日一句话建议
├── 当前表现摘要
├── 主要弱项
├── 原因分析
├── 下一步行动
├── 学习节奏建议
└── 鼓励反馈
```

### 5.3 顶部文案

```text
你的私人英语教练
我会根据你的单词和阅读表现，给出下一步学习建议。
```

### 5.4 今日一句话建议

#### 有数据状态

```text
今日建议
先复习 4 个薄弱词，再做一篇包含推断题的阅读。
```

#### 数据不足状态

```text
今日建议
先完成一组单词和一篇阅读，我会据此生成更准确的个性化建议。
```

### 5.5 当前表现摘要

```text
当前表现
词汇：近期词掌握较稳定，但仍有 4 个高频模糊词。
阅读：整体正确率 75%，推断题表现偏弱。
学习节奏：连续学习 3 天，建议保持当前任务量。
```

### 5.6 主要弱项

```text
主要弱项
1. 推断题容易忽略转折和作者态度
2. context、assume 等词在阅读语境中理解不稳定
3. 阅读后复盘不足，错题没有充分回流到词汇复习
```

### 5.7 原因分析

```text
可能原因
你对单词的中文释义记忆较好，但在长句和段落逻辑中识别不够稳定。尤其遇到 however、although、rather than 这类逻辑信号时，容易只看局部句子。
```

### 5.8 下一步行动

```text
下一步行动
1. 复习 4 个薄弱词：context、assume、maintain、evaluate
2. 做 1 篇推断题强化阅读
3. 做题后重点复盘文章转折句

[开始薄弱词复习]
[开始推荐阅读]
```

### 5.9 学习节奏建议

```text
学习节奏
当前建议保持每天：
10—15 个单词
1 篇阅读
4 道题

如果连续两天阅读正确率低于 60%，建议降低新词数量，优先复习薄弱词。
```

### 5.10 鼓励反馈

```text
今天你已经完成了输入和输出两个环节。比单纯背词更重要的是，你开始在文章里重新遇到这些词，这会让记忆更稳定。
```

### 5.11 无数据状态完整文案

```text
我还不够了解你的学习状态

完成以下两步后，我会生成第一份个性化建议：
1. 学习或复习至少 5 个单词
2. 完成 1 篇阅读理解

[开始今日学习]
```

### 5.12 交互规则

| 用户操作 | 页面反馈 |
|---|---|
| 点击开始薄弱词复习 | 进入单词页，并使用薄弱词作为本轮内容 |
| 点击开始推荐阅读 | 进入阅读页，并使用建议题型或词汇生成阅读 |
| 学习记录不足 | 显示入门建议，不生成伪个性化结论 |
| 用户完成新任务后返回 | 更新当前表现和下一步行动 |

## 6. 关键边界文案

### 6.1 AI 生成内容提示

```text
AI 生成内容可能存在不准确之处。请以题目解析和原文依据为参考，如发现明显问题，可以重新生成或更换练习。
```

### 6.2 个性化建议边界

```text
这些建议基于你在平台内的近期学习记录，不代表正式考试成绩预测。
```

### 6.3 学习效果边界

```text
持续完成单词和阅读练习有助于巩固记忆，但具体提升效果取决于学习频率、复盘质量和个人基础。
```
