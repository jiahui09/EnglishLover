# UI Freedom Policy

本项目允许 Gemini 充分发挥 UI 设计能力。

采用：

> 按需复用功能，允许重新设计风格。

现有组件是功能资产，不是视觉模板。

允许：

- 重新设计页面视觉语言。
- 重新设计导航、布局、背景、卡片、状态提示。
- 使用现有组件的功能和语义，但改造其视觉呈现。
- 新增页面级样式。
- 扩展视觉 token。
- 调整 layout 组件外观。
- 对组件做不破坏 props 的样式增强。

不要求：

- 不要求固定使用某个组件清单。
- 不要求沿用现有组件默认外观。
- 不要求保留当前白板风格。

仍需遵守：

- 不破坏组件功能语义。
- 不改 API 类型。
- 不造假数据。
- 不新增依赖。
- 不展示工程穿帮文案。
