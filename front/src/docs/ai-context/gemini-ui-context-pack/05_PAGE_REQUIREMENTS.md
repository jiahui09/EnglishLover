# Page Requirements

## `/today`

目标：今日学习工作台。

必须包含：

- 今日学习标题或问候。
- 单词、阅读、写信三个核心任务入口。
- 今日进度。
- 最近提醒或继续上次。
- 加载态、空态、错误态。
- 明确主行动。

## `/vocabulary`

目标：专注单词练习。

必须包含：

- 当前复习状态。
- 练习题区域。
- 答题反馈。
- 复习进度。
- 无待复习内容时的空态。
- 加载失败时的重试。

## `/reading`

目标：舒适阅读和答题。

必须包含：

- 阅读文章区域。
- 阅读进度。
- 阅读题区域。
- 生词操作。
- 加载态、空态、错误态。
- 阅读友好的行宽和留白。

## `/penpal`

目标：安静、安全的写信空间。

必须包含：

- 线程或关系状态。
- 写信编辑器。
- 自动保存状态。
- 安全检查状态。
- 安全提示。
- 问题高亮。
- 无可写对象时的解释性空态。

## `/results`

目标：轻量成果回顾。

必须包含：

- 今日成果摘要。
- 模块完成情况。
- 进度或趋势区域。
- 无成果时返回今日学习的入口。
- 不得用假统计制造完成感。
